# Keyboard Shortcuts

The Brand Kit Dashboard supports several keyboard shortcuts for quick access to features:

## Global Shortcuts

- **Ctrl/Cmd + K**: Open settings page
- **Escape**: Close any open modal/dialog

## Quick Add Shortcuts

These shortcuts work when you're not typing in an input field:

- **R**: Add a new reminder
- **L**: Add a new link
- **N**: Add a new note

## Tips

- All shortcuts are case-insensitive
- Shortcuts are disabled when typing in input fields or textareas
- Use Escape to quickly dismiss any modal that's open
